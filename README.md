# mongodb-todo-app

## GET:

Endpoint | Function
--------- | ----------
/todos/ | List all Todos
/todos/>id</| List a single Todo

## POST:

Endpoint | Function
--------- | ---------
/users/ | Register a user
/users/login/ | Login a user
/todos/ | Create a Todo

## PATCH:

Endpoint | Function
--------- | ----------
/todos/>id< | Update an existing todo

## DELETE:

Endpoint | Function
--------- | ----------
/todos/>id< | Delete an existing todo
